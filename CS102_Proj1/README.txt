Project 1
---------
To run, use: 
javac MainFindWords.java
java MainFindWords dictionary_80000.txt

The purpose of this program is to reorder letters (from a given set of letters) to create words.  
A dictionary is loaded from a text file and the program uses it to determine if a given set of letters is a word.
For example if the given letters are: 
"atrac"
The program would print:
(It prints the results in alphabetical order)
acta
arcat
at
car
carat
carta
cat
catar
rat
ta
